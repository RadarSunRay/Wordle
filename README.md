Wordle (Console)

Мой проект, сделанный на основе игры угадывания слов из 5 пяти букв на английском языке (C#).

## 🚀 Особенности
- Реализована основная логика игры с угадыванием слова.
- Отображение букв ответа по клеткам.
- Присвоение клеткам соответствующие цвета.
- Отображение алфавита, хранящего в себе неправильные буквы.
- Использование ООП.
- Использование методов с рамками и точками.

## 🛠 Технологии
- Язык: C#
- Платформа: .NET 10.0 / Console Apps

## 📈 Чему я научился в этом проекте
1. Работа с символами (`char`).
2. Работа с цветами (`Console.ForegroundColor`).
3. Работа с LINQ.
4. Работа с проверками на ввод (`ConsoleKeyInfo`).

## Как запустить
1. Склонируйте репозиторий: `git clone https://github.com/RadarSunRay/Wordle.git`
2. Откройте решение `.sln` в Visual Studio.
3. Нажмите `F5` для запуска.
